## 🎯 วัตถุประสงค์ (Project Goals)
* ฝึกฝนการใช้งานคำสั่ง `print()` ในการจัดการ String
* เรียนรู้การใช้ Multi-line strings (`'''`) สำหรับการจัดรูปแบบข้อความขนาดใหญ่
* สร้างพื้นฐานการจัดเก็บ Code และ Documentation บน GitHub

## 💻 ชุดโค้ดตัวอย่าง (The Implementation)

```python
# แสดงข้อความทักทายโลก
print('Hello World')

# ศิลปะเรือใบที่สร้างจากอักขระ ASCII
text = '''
           |    |    |
          )_)  )_)  )_)
         )___))___))___)
        )____)____)____)
      _____|____|____|______
      \                    /
~~~~~~~ ~~~~~~~ ~~~~~~~ ~~~~~~~ ~~~~~~~
'''

# แสดงผลกราฟิกเรือใบออกมาทางหน้าจอ Console
print(text)# helloworld
